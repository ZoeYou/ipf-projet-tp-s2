This package contains:

 * the executable file: main
 * the program source code files: ants.ml main.ml
 * the ants.mli header file that define the Ocaml-language
   interface to the ants.ml library file
 * a Makefile.msc for building

 SUMMARY OF HOW TO BUILD
=======================

  Unix:      ./configure; make
  Windows:   nmake /f Makefile.msc

AFTER INSTALLATION
===================

If you success to build, you will see the program runs as below:
---------------------------------------------------------------

Welcome to the extraterrestrial message-sending system! Choose:
q: to quit the system
p: to send a message

----------------------------------------------------------------
Then you can start using the extraterrestrial message-sending system,
hope you will have a good experience with it!
